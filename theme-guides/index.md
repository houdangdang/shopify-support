# Theme Specific Guides

Detailed guides for each Maestrooo theme.

## Our Themes

Choose your theme to see specific setup guides and features:

### [Prestige](/theme-guides/prestige)
Perfect for luxury and high-end brands. Features:
- Elegant animations
- Premium product galleries
- Editorial-style layouts

### [Impact](/theme-guides/impact)
Bold, visual-first design. Features:
- Full-width imagery
- High-impact sections
- Video backgrounds

### [Focal](/theme-guides/focal)
Designed for photography and creative stores. Features:
- Gallery-focused layouts
- Lightbox displays
- Minimalist design

### [Warehouse](/theme-guides/warehouse)
Built for large catalogs. Features:
- Advanced filtering
- Quick view/quick add
- Bulk product display

### [Stretch](/theme-guides/stretch)
Flexible multi-purpose theme. Features:
- Extensive customization
- Multiple layout options
- Versatile sections
