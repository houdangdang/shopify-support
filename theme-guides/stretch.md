# Stretch Theme

A versatile, multi-purpose theme for any store type.

## Overview

Stretch adapts to any brand:
- Extensive customization
- Multiple layout options
- Versatile sections
- Flexible design

## Unique Features

### Layout Flexibility
- Switchable layouts
- Grid customization
- Responsive controls

### Section Variety
- 25+ section types
- Custom blocks
- Drag-and-drop reordering

### Style Options
- Multiple header styles
- Various card layouts
- Font combinations

## Quick Setup

1. Choose your layout style
2. Configure header/footer
3. Add sections to homepage
4. Customize colors and fonts
