# Focal Theme

A minimalist theme for photography and creative stores.

## Overview

Focal puts your imagery center stage:
- Clean, minimal design
- Gallery-focused layouts
- Whitespace emphasis
- Photography-first approach

## Unique Features

### Image Galleries
- Lightbox displays
- Slideshow layouts
- Masonry grids

### Minimal UI
- Hidden chrome
- Focus on content
- Clean typography

### Creative Layouts
- Asymmetrical sections
- Overlapping elements
- Image collages

## Quick Setup

1. High-quality product photography
2. Configure gallery lightbox
3. Set minimal header style
4. Use generous whitespace
