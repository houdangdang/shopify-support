# Impact Theme

A bold, visual-first theme for impactful brands.

## Overview

Impact creates striking visual experiences:
- Full-width imagery
- Bold typography
- Video backgrounds
- High-contrast design

## Unique Features

### Video Hero
- Full-screen video backgrounds
- Autoplay with controls
- Mobile-optimized fallback images

### Split Sections
- 50/50 image and text layouts
- Animated reveals
- Overlapping elements

### Bold Typography
- Large display fonts
- Impactful headlines
- Call-to-action emphasis

## Quick Setup

### 1. Homepage
1. Add **Video hero** section
2. Use **Split sections** for content
3. Add **Full-width image** sections

### 2. Visual Settings
1. Enable animations
2. Configure color contrast
3. Set up typography scale

### 3. Mobile Optimization
1. Set video fallback images
2. Adjust mobile typography
3. Test touch interactions

## Theme-Specific Settings

| Setting | Location | Options |
|---------|----------|---------|
| Video autoplay | Section settings | Yes, No, Muted |
| Text contrast | Theme settings | High, Medium, Low |
| Animation style | Theme settings | Fade, Slide, Scale |
