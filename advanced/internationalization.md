# Internationalization

Setup multi-language and multi-currency support.

## Languages

Our themes support multiple languages through Shopify's translation system.

### Adding Languages

1. Go to **Settings** → **Languages**
2. Click **Add language**
3. Choose languages
4. Publish when ready

### Translation Methods

| Method | Best For |
|--------|----------|
| Translate & Adapt app | DIY translation |
| Third-party apps | Professional translation |
| CSV export/import | Bulk editing |

## Currencies

### Enable Multi-Currency

1. Go to **Settings** → **Payments**
2. Manage currencies
3. Enable desired currencies

### Currency Display

Theme shows:
- Currency selector in header/footer
- Converted prices
- Local currency formatting

## Markets

Shopify Markets for international selling:

1. Create markets for regions
2. Set pricing strategies
3. Configure shipping
4. Customize domains

## RTL Support

Right-to-left language support:
- Arabic
- Hebrew
- Persian

Theme automatically adjusts layout for RTL languages.

::: tip SEO for International
Use hreflang tags (automatically added) for international SEO.
