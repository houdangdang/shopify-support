# Using Apps

Integrate third-party apps with your theme.

## App Compatibility

Our themes are compatible with most Shopify apps:
- Product reviews
- Wishlist apps
- Loyalty programs
- Email marketing
- Chat widgets

## App Blocks

Modern themes support app blocks:

1. Install an app with theme app extensions
2. Open theme editor
3. Add app blocks to sections
4. Configure and position

## Common Integrations

### Reviews Apps
- Judge.me
- Loox
- Stamped.io
- Yotpo

### Marketing Apps
- Klaviyo
- Privy
- Smile.io

### Functionality Apps
- Recharge (subscriptions)
- Bold (bundles)
- PageFly (landing pages)

## Troubleshooting

| Issue | Solution |
|-------|----------|
| App not showing | Check app block placement |
| Styling conflicts | Add custom CSS |
| Performance issues | Contact app developer |

::: info App Support
Issues caused by third-party apps should be directed to the app developer.
:::
