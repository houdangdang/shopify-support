# Collection Configuration

Customize collection page layouts and settings.

## Layout Options

Choose your collection layout:

| Layout | Description |
|--------|-------------|
| Grid | Product grid with filters |
| List | Product list view |
| Masonry | Pinterest-style grid |

## Products Per Row

Configure grid columns:
- Desktop: 3, 4, or 5 products
- Tablet: 2 or 3 products
- Mobile: 1 or 2 products

## Collection Banner

Customize the collection header:

| Setting | Options |
|---------|---------|
| Image | Collection image or custom |
| Height | Small, Medium, Large |
| Text position | Left, Center, Right |
| Overlay | Color and opacity |

## Show/Hide Elements

- [ ] Product count
- [ ] Sort dropdown
- [ ] View toggle (grid/list)
- [ ] Filters
- [ ] Subcollection links

## Pagination

Choose pagination style:
- **Traditional**: Page numbers
- **Load more**: Button to load more
- **Infinite scroll**: Auto-load on scroll

Configure products per page: 12, 24, 36, 48
