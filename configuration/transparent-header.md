# Transparent Header

Create a transparent header that overlays hero images.

## When to Use

Transparent headers work best with:
- Full-width hero images
- Slideshow sections
- Video backgrounds
- Image banners

## Enable Transparent Header

1. Open the theme editor
2. Select the **Header** section
3. Find **Transparent header** settings
4. Enable the toggle

## Configuration Options

### Text Color
Set the text/icon color when transparent:
- Light (for dark backgrounds)
- Dark (for light backgrounds)

### Logo Override
Upload a different logo for transparent mode:
- Use light logo on dark images
- Use dark logo on light images

### Border
Optional border below transparent header:
- Color
- Width
- Style

## Page-Specific Settings

Control which pages show transparent header:

```
Home page: ✓ Enabled
Collection pages: ✗ Disabled
Product pages: ✗ Disabled
```

## Best Practices

::: tip Ensure Readability
- Choose contrasting text color
- Add text shadow if needed
- Test with all slideshow images
:::

::: warning Avoid Conflicts
Don't use transparent header with:
- Light hero images + Dark text
- Busy/patterned backgrounds
- Low contrast images
:::

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Text not visible | Change text color or add overlay |
| Logo too dark | Upload transparent mode logo |
| Header overlaps content | Check hero section padding |
