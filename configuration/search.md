# Search Configuration

Optimize your store's search experience.

## Search Types

| Type | Description |
|------|-------------|
| **Predictive** | Live results as you type |
| **Full page** | Dedicated search results page |

## Predictive Search

Configure in **Header** section:

### Show in Results
- Products (images, prices)
- Collections
- Pages/Articles
- Suggestions

### Display Options
- Number of results
- Show product images
- Show prices
- Show vendor

## Search Results Page

Configure layout and filters on collection template.

## Search Behavior

### What's Searchable
- Product titles
- Product descriptions
- Product tags
- SKUs
- Vendor names
- Metafield content

### Search Operators
Customers can use:
- `AND` - Both terms required
- `OR` - Either term
- `-` - Exclude term
- `""` - Exact phrase

## Configuration

In theme editor:

| Setting | Options |
|---------|---------|
| Placeholder text | Custom text |
| Show trending | Recent searches |
| Show suggestions | Popular queries |
| Minimum characters | Start search after X characters |
