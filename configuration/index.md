# Configuration

Configure your theme settings for the perfect store experience.

## General Settings

Access theme-wide settings through **Theme settings** in the editor:

| Setting | Description |
|---------|-------------|
| **Colors** | Brand colors, backgrounds, text |
| **Typography** | Font families and sizes |
| **Layout** | Container widths, spacing |
| **Buttons** | Button styles and hover states |
| **Social Media** | Social profile links |
| **Favicon** | Browser tab icon |
| **Custom CSS** | Additional styling |

## Section Configuration

Each section has its own settings:

### Common Section Settings
- Background color/image
- Padding and spacing
- Container width
- Mobile visibility

### Block Settings
Sections contain blocks that can be:
- Added or removed
- Reordered via drag-and-drop
- Individually configured
