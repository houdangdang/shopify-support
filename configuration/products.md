# Product Configuration

Configure how products are displayed across your store.

## Product Card Settings

Customize product cards in collections and grids:

| Setting | Options |
|---------|---------|
| Image ratio | Square, Portrait, Landscape |
| Show second image on hover | Yes/No |
| Show vendor | Yes/No |
| Show rating | Yes/No |
| Show quick add | Yes/No |

## Product Page Layout

Choose your product page layout:

- **Layout 1**: Gallery left, info right
- **Layout 2**: Stacked layout
- **Layout 3**: Gallery right, info left
- **Layout 4**: Full-width gallery

### Gallery Options

| Setting | Description |
|---------|-------------|
| Gallery type | Thumbnails, Dots, Slideshow |
| Zoom type | Lightbox, Hover zoom, Click zoom |
| Video autoplay | Auto, Click to play |

## Variant Display

Configure how variants are shown:

- **Buttons**: Clickable variant buttons
- **Dropdown**: Select dropdown
- **Color swatches**: Visual color options

See [Color Swatches](/configuration/color-swatches) for details.

## Related Products

Configure product recommendations:

| Setting | Options |
|---------|---------|
| Show related products | Yes/No |
