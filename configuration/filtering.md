# Products Filtering

Configure collapsible filters for collection pages.

## Filter Types

Shopify provides built-in filter types:

| Filter | Description |
|--------|-------------|
| Availability | In stock / Out of stock |
| Price | Price range slider |
| Product type | Product categories |
| Vendor | Brand/manufacturer |
| Options | Size, Color, etc. |
| Metafields | Custom attributes |

## Enable Filters

1. Go to **Online Store** → **Navigation**
2. Click **Collection and search filters**
3. Add desired filters
4. Arrange order

## Theme Configuration

In theme editor → Collection page:

### Filter Display

| Setting | Options |
|---------|---------|
| Style | Sidebar, Horizontal, Drawer |
| Collapse by default | Yes/No per filter |
| Show count | Number of products |

### Filter Behavior

- **Apply immediately** or **Apply button**
- **Multi-select** for options
- **Price range** with inputs

## Color Filters as Swatches

Display color options visually:

1. Enable **Show as swatches** for color filter
2. Configure swatch colors in theme settings
3. Matches [Color Swatches](/configuration/color-swatches) setup

## Mobile Filters

Configure mobile filter behavior:

- Sticky filter bar
- Full-screen filter drawer
- Filter count badge

::: tip Performance
Too many filters can slow page load. Enable only necessary filters.
:::
