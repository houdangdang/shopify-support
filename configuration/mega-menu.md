# Mega Menu

Create rich navigation menus with images and multiple columns.

## Overview

Mega menus allow you to create visually rich dropdown menus with:
- Multiple columns of links
- Featured images
- Promotional banners
- Collection highlights

## Setup Steps

### 1. Create Navigation Menu

1. Go to **Online Store** → **Navigation**
2. Create or edit your main menu
3. Add top-level items (these become mega menu triggers)
4. Add nested items under each top-level item

### 2. Enable Mega Menu

1. Open the theme editor
2. Go to **Header** section
3. Find **Navigation style**
4. Select **Mega menu**

### 3. Configure Mega Menu

For each top-level item, you can configure:

| Setting | Options |
|---------|---------|
| Layout | Single column, Multi-column, Featured |
| Image | Featured product or collection image |
| Columns | 2, 3, 4, or 5 columns |
| Promo block | Optional promotional content |

## Adding Images

To add images to your mega menu:

1. In **Theme settings** → **Header**
2. Find **Mega menu images**
3. Upload images
4. Link to the matching menu item name

::: tip Menu Item Names
The image is matched by the menu item's exact text. For "Women's Clothing", name your image setting "Women's Clothing".
:::

## Styling Options

- Background color
- Text color
- Padding and spacing
- Border styles
- Animation effects

## Mobile Behavior

On mobile devices, the mega menu converts to a collapsible accordion menu for easier navigation.
