# Header Configuration

Customize your store's header and navigation.

## Header Settings

Access header settings: **Header** section in theme editor.

### Layout Options

| Layout | Description |
|--------|-------------|
| **Logo left** | Logo on left, nav in center |
| **Logo center** | Centered logo, nav below |
| **Logo left inline** | Logo and nav on same line |

### Header Elements

Configure visibility of:
- Logo
- Search bar
- Cart icon
- Account link
- Currency/language selector
- Announcement bar

## Sticky Header

Enable a header that stays visible while scrolling:

1. Open **Header** section
2. Enable **Sticky header**
3. Choose behavior:
   - Always sticky
   - Sticky on scroll up
   - Hide on scroll down

## Transparent Header

Make the header transparent over hero images:

1. Open **Header** section
2. Enable **Transparent header**
3. Configure:
   - Text color on transparent
   - Logo on transparent (optional)

See [Transparent Header](/configuration/transparent-header) for details.

## Search Configuration

| Setting | Options |
|---------|---------|
| Search type | Predictive, Full page |
| Show product images | Yes/No |
| Show collections | Yes/No |
| Show pages | Yes/No |

## Mobile Header

Mobile-specific settings:
- Hamburger menu position
- Mobile logo size
- Drawer menu style
