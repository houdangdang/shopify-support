# Getting Started

Welcome to the GirlElements Themes documentation. This guide will help you get started with your theme and make the most of its features.

## About GirlElements Themes

GirlElements creates premium Shopify themes that are designed to help you build beautiful, high-converting online stores. Our themes are:

- **Fully customizable** - Extensive theme settings and section options
- **Mobile responsive** - Optimized for all devices
- **Performance focused** - Fast loading times for better SEO
- **Accessible** - Built with accessibility best practices
