# Accessibility (ADA, a11y)

Our themes are built with accessibility in mind.

## WCAG 2.1 Compliance

We strive to meet WCAG 2.1 Level AA standards:

### Perceivable
- Text alternatives for images
- Captions and transcripts where applicable
- Sufficient color contrast
- Resizable text up to 200%

### Operable
- Keyboard navigation support
- No time limits on interactions
- Skip navigation links
- Clear focus indicators

### Understandable
- Consistent navigation
- Clear error messages
- Predictable functionality

### Robust
- Valid HTML markup
- Compatible with assistive technologies
- ARIA attributes where needed

## Testing Tools

We test with:

- **Screen readers** - VoiceOver, NVDA, JAWS
- **Keyboard navigation** - Tab, Enter, Space, Arrow keys
- **Color contrast analyzers** - Ensuring 4.5:1 ratio for text
- **Automated tools** - Lighthouse, axe, WAVE

## Customization Tips

::: tip Maintain Accessibility
When customizing your theme:
- Ensure color changes maintain contrast ratios
- Don't remove focus indicators
- Keep alt text on images
- Test with keyboard navigation
:::
