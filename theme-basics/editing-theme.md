# Editing Your Theme

Learn how to customize your theme using the Shopify theme editor.

## Accessing the Theme Editor

1. Go to your Shopify admin
2. Navigate to **Online Store** → **Themes**
3. Click **Customize** on your current theme

![Theme Editor Interface](/assets/images/theme-basics/theme-editor-sections.png)

## Theme Editor Overview

The theme editor consists of:

### Left Panel - Sections
- Add, remove, and reorder sections
- Configure section settings
- Manage blocks within sections

### Center - Preview
- Live preview of your changes
- Device viewport switcher (desktop/mobile)

### Top Bar
- Page selector (Home, Product, Collection, etc.)
- Undo/Redo buttons
- Save button

## Global Theme Settings

Click the **Theme settings** icon (gear) to access:

![Theme Settings Icon](/assets/images/theme-basics/theme-settings-icon.png)

| Setting | Description |
|---------|-------------|
| Colors | Brand colors, text, backgrounds |
| Typography | Fonts and sizes |
| Layout | Container widths, spacing |
| Buttons | Button styles and colors |
| Social media | Social links |
| Favicon | Browser tab icon |

## Adding Sections

1. Click **Add section** in the left panel
2. Choose a section type
3. Configure the section settings
4. Click **Save**

::: tip Section Limits
Some sections can only be added once per page, while others can be added multiple times.
:::

## Saving Changes

- Click **Save** to publish changes immediately
- Use **Undo** if you make a mistake
- Preview on different devices before saving

## Custom CSS

For advanced styling, add custom CSS in:
**Theme settings** → **Custom CSS**

```css
/* Example: Change header background */
.header {
  background-color: #000000;
}
```
