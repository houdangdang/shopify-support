# Install and Activate the Theme

This is the guideline to Install and Activate the theme:

## Unzip the Theme package
After you bought our theme, please extract the download package which you just downloaded from Theme Forest: https://themeforest.net/downloads

<img src="/assets/images/theme-basics/extract-file.png" alt="" style="max-height: 300px; width: auto; display: block; border: 1px solid #ccc; border-radius: 4px;">

Then you will see the upload file, please upload the xxx.zip.

<img src="/assets/images/theme-basics/upload-zip.png" alt="" style="max-height: 300px; width: auto; display: block; border: 1px solid #ccc; border-radius: 4px;">

## How to install the theme?
**Step 1**: From the Shopify backend, click on **Online Store > Themes > Import theme > Upload zip file**.

Choose the **Upload zip file** button:

<img src="/assets/images/theme-basics/how-to-upload-zip.png" alt="" style="max-height: 300px; width: auto; display: block; border: 1px solid #ccc; border-radius: 4px;">

**Step 2**: On the Upload theme popup, click **Add file >  Upload Theme here > Press the Upload file button**:

<img src="/assets/images/theme-basics/select-zip.png" alt="" style="max-height: 400px; width: auto; display: block; border: 1px solid #ccc; border-radius: 4px;">

**Step 3**: After a minute, the theme is uploaded, you can click **Edit theme** button to activate the theme and start building your store:

<img src="/assets/images/theme-basics/edit-theme.png" alt="" style="max-height: 300px; width: auto; display: block; border: 1px solid #ccc; border-radius: 4px;">

## How to activate the theme?
**Step 01**: In Customize Theme backend, go to Theme Settings to activate the theme.

**Step 02**: In the Theme Active option, paste your license code in the Purchase code option. Then click the Save button to activate the theme.

**Step 03**: Reload the page, the theme has been activated.

<img src="/assets/images/theme-basics/activate-theme.gif" alt="" style="max-height: 600px; width: auto; display: block; border: 1px solid #ccc; border-radius: 4px;">
