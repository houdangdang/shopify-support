# Image Size Recommendations

Optimal image dimensions for best quality and performance.

## General Guidelines

- Use **JPG** for photos
- Use **PNG** for graphics with transparency
- Use **WebP** for best compression (auto-converted by Shopify)

## Recommended Sizes

### Product Images

| Image Type | Recommended Size | Aspect Ratio |
|------------|------------------|--------------|
| Main product | 2048 x 2048 px | 1:1 (square) |
| Product gallery | 2048 x 2048 px | 1:1 |
| Zoom enabled | 4000 x 4000 px | 1:1 |

### Collection Images

| Image Type | Recommended Size |
|------------|------------------|
| Collection banner | 2400 x 800 px |
| Collection thumbnail | 1200 x 1200 px |

### Section Images

| Section | Recommended Size |
|---------|------------------|
| Slideshow | 2400 x 1000 px |
| Image with text | 1200 x 800 px |
| Logo | 400 x 200 px |
| Favicon | 32 x 32 px |

### Blog Images

| Image Type | Recommended Size |
|------------|------------------|
| Featured image | 1600 x 900 px |
| Article body | 1200 x auto |

## Performance Tips

::: tip Optimize Images
- Compress images before uploading
- Use consistent aspect ratios
- Avoid images larger than 4000px
:::

### Compression Tools

- [TinyPNG](https://tinypng.com)
- [Squoosh](https://squoosh.app)
- [ImageOptim](https://imageoptim.com) (Mac)

## Retina Displays

Shopify automatically serves 2x images for retina displays. Upload at recommended sizes for best quality.
