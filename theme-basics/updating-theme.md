# Updating Your Theme

How to safely update your theme to the latest version.

## Before Updating

::: warning Backup First
Always download a backup of your current theme before updating.
:::

### Create a Backup

1. Go to **Online Store** → **Themes**
2. Click **Actions** → **Download theme file**
3. Save the ZIP file safely

### Document Customizations

Note any custom code you've added:
- Custom CSS
- Modified Liquid files
- Custom sections
- JavaScript changes

## Update Methods

### Method 1: Theme Store Update

If you purchased from the Shopify Theme Store:

1. You'll see an **Update available** badge
2. Click **Update**
3. Review changes
4. The update creates a new theme copy

![Theme Update Available](/assets/images/getting-started/theme-update.png)

### Method 2: Manual Update

1. Download the latest theme version
2. Upload as a new theme
3. Copy your customizations:
   - Theme settings
   - Custom CSS
   - Section configurations
4. Test thoroughly
5. Publish when ready

## After Updating

### Test Your Store

Check these pages:
- [ ] Homepage
- [ ] Product pages
- [ ] Collection pages
- [ ] Cart and checkout
- [ ] Mobile views

### Re-apply Customizations

1. Copy custom CSS to the new theme
2. Reconfigure section settings
3. Check metafield connections
4. Verify navigation menus

## Common Issues

| Issue | Solution |
|-------|----------|
| Settings lost | Re-configure in theme editor |
| Custom code missing | Copy from backup |
| Layout broken | Check for deprecated sections |

::: info Need Help?
Contact [support](/contact) if you encounter issues after updating.
